# NFTMarketHub

[![Deploy to GitHub Pages](https://github.com/yourname/NFTMarketHub/actions/workflows/deploy.yml/badge.svg)](https://github.com/yourname/NFTMarketHub/actions)

多语言NFT市场导航网站，支持13种语言。

## 功能特性
- 实时市场数据展示
- 多语言切换（中/英/西等）
- 自动每日数据更新
- SEO优化

## 部署步骤
1. 替换配置文件：
   - `CNAME` 设置您的域名
   - `analytics.js` 添加GA跟踪ID
2. 上传到GitHub仓库
3. 启用GitHub Pages

## 数据更新
每日自动通过：
- CoinGecko API
- DappRadar 数据
- 手动维护的精选平台列表
